# AI expansion

Stub — planned for Phase 15.

Principles:

- AI is **opt-in** and **bring-your-own-key**. RepoPulse never makes AI calls in the default configuration.
- AI augments — it never replaces — the deterministic checks.
- Users see what content would be sent before any call is made.
- Costs are surfaced.
- A no-AI mode is always available.
